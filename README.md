# React Portofolio v3

Build and deploy a responsive React Portfolio Website with a light & dark mode, a cartoon hero image of me generated with AI.

## Demo:

https://pedrogomesportofolio.netlify.app

## Install npm Packages:

#### `npm install`

## Run App Locally:

#### `npm run dev`

## Deploy App:

#### `npm run build`

#### `npm run preview`
